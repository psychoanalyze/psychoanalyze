# Tooling

Psychoanalyze uses

- Python 3.11
- [Poetry](https://python-poetry.org/) for package and virtual environment management
- [Pytest](https://docs.pytest.org/) for automated testing
- [MkDocs](https://www.mkdocs.org/) for documentation static site generation
- [Heroku](https://www.heroku.com/) for dashboard deployment

[Dev Containers](https://containers.dev/) are the suggested way to set up a development environment. Running a dev container locally requires Docker Desktop on OSX and Windows.
