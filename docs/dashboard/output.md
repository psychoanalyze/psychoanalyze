## Output Panel

The Output Panel summarizes fitted psychometric curves and the aggregated trial data for the active dataset.

### Blocks

The Blocks table lists per-block logistic fit parameters (intercept and slope). Select one or more blocks to filter the points table and the plotted fits.

### Points

The Points table aggregates trials by intensity level, showing trial counts, hits, and hit rates.

### Downloads

Use the download links to export the current plot as SVG, PNG, or PDF, and to export the active data as CSV (zip), JSON, Parquet, or DuckDB.
