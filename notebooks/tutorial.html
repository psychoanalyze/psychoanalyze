1.1.1 Data Schema

1.1.2 Diving into Logistic Regression

By designing the experiment such that the recorded response is binary, researchers may use logistic regression to model the relationship between the stimulus and the response; logistic regression is a generalized linear model that is appropriate for a series of binary (Bernoulli) trials.

Let’s examine the logistic function using built-in psychoanalyze utility functions.

First, let’s plot the standard logistic equation:

from psychoanalyze.data.blocks import standard_logistic
from psychoanalyze import plot
import pandas as pd
pd.options.plotting.backend = "plotly"

plot.standard_logistic()

\[ f(x) = \frac{1}{1 + e^{-x}} \]

Sanity check

\(f(0) = \frac{1}{1 + e^{-0}} = \frac{1}{1+1} = \frac{1}{2}\)

1.1.3 Apply the logistic model to a Yes/No experiment

Let’s label some variables and perform some simple substitutions to examine a non-standard parameterization of the logistic function, where is a location parameter and is a scale parameter:

\[ \psi(x) = \frac{1}{1 + e^{-\frac{x - \mu}{\sigma}}} \]

Observe how the x-axis adjusts in response to the following adjustments:

  • To the location parameter, 
plot.logistic(location=10, scale=1)
  • To the scale parameter, 
plot.logistic(location=0, scale=10)
